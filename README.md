## 🚀 Como Executar

### Pré-requisitos

- Node.js 18+ ou npm/yarn

### Instalação

```bash
# Clonar repositório
git clone https://github.com/wgutto/augusto-brito-portfolio
cd augusto-brito-portfolio

# Instalar dependências
npm install
```

### Desenvolvimento

```bash
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no navegador.
